# macOS — System Settings · Appearance

The **paper** side of the system: a macOS System Settings panel applying the
dotmesh language to native chrome. Light surfaces, hairline borders, the syntax
family offered as selectable accent swatches.

- `index.html` — entry. Loads the compiled `_ds_bundle.js`.
- `SystemSettings.jsx` — composes the design-system primitives
  (`Card`, `Switch`, `Button`, `Badge`, `Input`) from
  `window.DotmeshDesignSystem_512187`, plus a few local layout pieces (nav,
  segmented control, swatches).

This kit demonstrates the components in a real screen, so it requires the bundle
(generated by the compiler). Interactions: switch theme, pick an accent, toggle
surface options, sync per-tool.
